System-wide touch simulation for iOS.

ZXTouch Rootless is an open-source automation framework that simulates touch events at the system level — no app injection, no tweaks per app. Script it from any language over a socket, record & replay gestures, match images, and run OCR. Now fully ported to rootless jailbreaks.

Capabilities

Everything you need to automate iOS.

One tweak, one socket, a full automation toolkit. Every capability works across the whole OS — home screen, any app, any game.

True Multitouch

Drive up to 19 simultaneous fingers with independent down / move / up phases. Pinch, rotate, and complex game gestures are first-class.

Script in Any Language

A clean socket protocol on port 6000 plus a batteries-included Python module. If it can open a TCP connection, it can automate your device.

Recording & Playback

Capture real touch sessions and replay them perfectly — from the floating panel, the Python API, or iOS Shortcuts.

Image Matching

Template matching on the live screen, accelerated by Apple's Accelerate framework. Find a button, tap it — no fragile coordinates.

On-Device OCR

Read text straight off the screen and branch your scripts on it. Built into the toolkit — no cloud, no screenshots leaving the device.

Shortcuts Integration

Trigger recorded scripts from iOS Shortcuts and automations — run automation on a schedule, an NFC tag, or a button press.

Shell Commands

Execute commands as root from your scripts with run_shell_command() — glue device state into your automation.

Low-Latency Remote

Control the device live from your computer or another device over the network — fast enough for real-time mirrored input.

Floating Control Panel

Double-click volume-down to summon an on-screen panel: run scripts, record touches, and manage settings anywhere in the OS.

Quick start

Install the tweak (see Installation), make sure Python 3 from the Procursus repo is present, then connect to the daemon listening on port 6000. The Python module ships with the tweak.

quickstart.py
from zxtouch.client import zxtouch
from zxtouch.touchtypes import *
import time

device = zxtouch("127.0.0.1")  # on-device, or your iPhone's IP remotely

device.touch(TOUCH_DOWN, 5, 400, 400)  # finger 5 down at (400, 400)
time.sleep(1)
device.touch(TOUCH_MOVE, 5, 400, 600)  # drag down 200pt
time.sleep(1)
device.touch(TOUCH_UP, 5, 400, 600)

device.disconnect()

Touch events

Every touch has a type (TOUCH_DOWN / TOUCH_MOVE / TOUCH_UP), a finger index (up to 19 concurrent), and screen coordinates. Batch multiple fingers in one frame with touch_with_list() — that's how pinch and multi-finger game controls work.

pinch.py
# two fingers moving apart = pinch-to-zoom
device.touch_with_list([
    {"type": TOUCH_DOWN, "finger_index": 1, "x": 350, "y": 600},
    {"type": TOUCH_DOWN, "finger_index": 2, "x": 450, "y": 700},
])
for i in range(20):
    device.touch_with_list([
        {"type": TOUCH_MOVE, "finger_index": 1, "x": 350 - i*8, "y": 600 - i*8},
        {"type": TOUCH_MOVE, "finger_index": 2, "x": 450 + i*8, "y": 700 + i*8},
    ])
    time.sleep(0.016)

Vision & OCR

Stop hardcoding coordinates. image_match() finds a template on the live screen (Accelerate-framework fast), and ocr() reads on-screen text so your scripts can react to real state.

find_and_tap.py
result = device.image_match("/var/mobile/button.png")
if result[0]:  # found on screen
    info = result[1]
    x = float(info["x"]) + float(info["width"])  / 2
    y = float(info["y"]) + float(info["height"]) / 2
    device.touch(TOUCH_DOWN, 1, x, y)
    device.touch(TOUCH_UP, 1, x, y)

Floating panel

Double-click volume down anywhere in iOS to open the ZXTouch control panel. From there you can run saved scripts, start/stop touch recording, and adjust settings — no computer required. Recordings are saved as ZXTouch scripts you can replay from the panel, Shortcuts, or play_script().

API reference zxtouch.client

MethodDescription
touch(type, finger, x, y)Simulate a single touch event — down, move, or up — for one finger index.
touch_with_list(events)Dispatch multiple touch events in one frame for true multitouch gestures.
image_match(path, ...)Template-match an image against the live screen; returns position & size.
ocr(region, ...)Run optical character recognition on a screen region.
pick_color(x, y)Return the RGB value of the pixel at the given coordinates.
start_touch_recording() / stop_touch_recording()Programmatically capture real touches into a replayable script.
play_script(path)Execute a recorded or written ZXTouch script.
switch_to_app(bundle_id)Bring any installed app to the foreground.
run_shell_command(cmd)Execute a shell command as root and capture its output.
insert_text(text) / move_cursor(offset)Type into and navigate the focused text field.
show_alert_box(title, msg, t) / show_toast(...)Display system alerts and toast notifications from scripts.
get_screen_size() / get_screen_orientation()Query display geometry for resolution-independent scripts.
get_device_info() / get_battery_info()Read device model, iOS version, battery level and state.
Examples

Copy, paste, automate.

auto_tapper.py — tap a spot every second
from zxtouch.client import zxtouch
from zxtouch.touchtypes import *
import time

device = zxtouch("127.0.0.1")
try:
    while True:
        device.touch(TOUCH_DOWN, 1, 390, 844)
        time.sleep(0.05)
        device.touch(TOUCH_UP, 1, 390, 844)
        time.sleep(1)
finally:
    device.disconnect()
app_pilot.py — launch an app & type
from zxtouch.client import zxtouch
import time

device = zxtouch("127.0.0.1")

device.switch_to_app("com.apple.MobileSMS")
time.sleep(2)
device.insert_text("sent by zxtouch ✨")
device.show_toast(1, "Message typed!", 1.5)

device.disconnect()
watcher.py — wait for a button, then tap
from zxtouch.client import zxtouch
from zxtouch.touchtypes import *
import time

device = zxtouch("127.0.0.1")

while True:
    ok, info = device.image_match("/var/mobile/claim.png")
    if ok:
        x = float(info["x"]) + float(info["width"]) / 2
        y = float(info["y"]) + float(info["height"]) / 2
        device.touch(TOUCH_DOWN, 1, x, y)
        device.touch(TOUCH_UP, 1, x, y)
        break
    time.sleep(0.5)
battery_guard.py — pause automation when hot
from zxtouch.client import zxtouch
import time

device = zxtouch("127.0.0.1")

ok, battery = device.get_battery_info()
if ok and float(battery["battery_level"]) < 0.2:
    device.show_alert_box("ZXTouch",
        "Battery under 20% — pausing script.", 3)
else:
    device.play_script("/var/mobile/farm.zxs")
Installation

Two minutes to first touch.

Grab the right .deb for your jailbreak from GitHub Releases (or the latest build from GitHub Actions), install, respring. Done.

Download .deb from Releases

Download the rootless package

From Releases, grab the file ending in _rootless.deb (built for Dopamine on iOS 16.4.1 / 16.6.1).

Install the deb

Use Sileo, Zebra, or install over SSH / NewTerm:

terminal
$ dpkg -i zxtouch_*_rootless.deb
$ killall -9 SpringBoard

Install Python 3

Add the Procursus repo in your package manager and install python3 — required to run automation scripts on-device.

Verify

Double-click volume-down — the floating panel should appear. The daemon is now listening on 127.0.0.1:6000.

Download the roothide package

From Releases, grab the file ending in _roothide.deb (built for Roothide / Serotonin on iOS 15.8 / 16.6.1).

Install the deb

Use your package manager or install from a terminal:

terminal
$ dpkg -i zxtouch_*_roothide.deb
$ killall -9 SpringBoard

Install Python 3

Install python3 from the Procursus repo to run scripts on-device.

Verify

Double-click volume-down for the floating panel. Daemon listens on 127.0.0.1:6000.
