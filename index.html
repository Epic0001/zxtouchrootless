Installation

Add the repo in Sileo or Zebra, or download the .deb directly.

Add Repo (Sileo / Zebra)

Recommended
  1. Open Sileo or Zebra
  2. Go to Sources → Add
  3. Enter the repo URL:
https://epic0001.github.io/zxtouchrootless
  1. Search for ZXTouch and install
  2. Install Python 3 from Procursus for script support

Manual Install

  1. Download the .deb for your jailbreak from Releases
  2. Choose _rootless.deb for Dopamine or _roothide.deb for Roothide
  3. Install via SSH:
dpkg -i <file>.deb && killall -9 SpringBoard

Python Scripts

  1. Add the Procursus repo in Sileo:
https://apt.procurs.us
  1. Install python3 from Procursus
  2. The zxtouch Python module is installed automatically with the .deb

Features

Everything you need to automate your iOS device at the system level.

👆

Touch Simulation

Multitouch support with up to 19 fingers. Record and replay touch sequences. System-level — works in any app.

🐍

Python Scripts

Write automation scripts in Python. Full output logging with timestamps. Run from the panel or via socket.

🔍

Image Matching

Find template images on screen using normalized cross-correlation. Built with Accelerate.framework — no OpenCV required.

🔤

OCR

Recognize text anywhere on screen using Vision framework. Returns text content with bounding box coordinates.

🎨

Color Tools

Pick the RGB value of any pixel. Search for colors in a screen region with tolerance ranges.

⌨️

Text Input

Insert text, show/hide keyboard, move cursor — via the appdelegate tweak injected into the frontmost app.

🔭

Siri Shortcuts

Every ZXTouch action is available as an iOS Shortcuts action. Automate without writing a single line of code.

🌙

Dark Mode

Toggle dark mode for both the app UI and the floating script panel from the Settings tab.

📦

Roothide Support

Native iphoneos-arm64e build for Roothide / Serotonin with proper dynamic path resolution via jbroot().

API Status

Connect to the tweak on port 6000 from Python or any language with socket support.

from zxtouch.client import zxtouch
from zxtouch.touchtypes import *
import time

device = zxtouch("127.0.0.1")  # use device IP for remote control

device.touch(TOUCH_DOWN, 1, 400, 400)
time.sleep(0.5)
device.touch(TOUCH_UP, 1, 400, 400)

result = device.ocr((0, 0, 1170, 2532))
for item in result[1]:
    print(item["text"], "at", item["x"], item["y"])

device.disconnect()
